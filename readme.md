Mini RAG Chatbot

Overview: This project implements a Retrieval-Augmented Generation (RAG) chatbot that answers questions from a PDF document using semantic search and vector embeddings.

Features
* PDF Text Extraction
* Text Chunking
* Embedding Generation using Sentence Transformers
* Vector Storage using FAISS
* Semantic Search
* Context Retrieval
* Question Answering

Technologies Used
* Python
* PyPDF
* Sentence Transformers
* FAISS
* NumPy
* Jupyter Notebook

Workflow
PDF Document
→ Text Extraction
→ Text Chunking
→ Embedding Generation
→ FAISS Vector Database
→ Semantic Search
→ Context Retrieval
→ Answer Generation

Sample Questions
* What is Artificial Intelligence?
* What is Machine Learning?
* What are the benefits of AI?
* What are the challenges of AI?
